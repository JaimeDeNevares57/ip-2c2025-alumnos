## Implementación
Para ejecutar la aplicación web primero es necesario ejecutar en la terminal del editor de código este comando: python -m http.server
Luego entrar al puerto http://localhost:8000/
### Fueron implementados:
- 3 algoritmos obligatorios (bubble, insertion, selection)
- 2 algoritmos opcionales (merge, quick)
- Menú de navegación
- 2 funcionalidades extras (además del ordenamiento)
- Métricas en la página principal
- Mejoras de experiencia para el usuario (visuales y música)
